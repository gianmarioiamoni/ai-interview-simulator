# Evaluation Pipeline

# Overview

The evaluation system is built around:

- executable validation
- structured evaluation
- semantic feedback
- adaptive retry decisions

---

# Core Pipeline

Answer
  ↓
Execution (optional)
  ↓
Evaluation
  ↓
Hint Generation
  ↓
Feedback Generation
  ↓
Decision
  ↓
Interview Aggregation
  ↓
Final Report

---

# Execution Layer

# Coding Questions

Uses:
- Python sandbox
- hidden tests
- visible tests

Execution outputs:
- passed tests
- failed tests
- execution time
- execution status

---

# SQL Questions

Uses:
- SQLite execution
- executable validation

- multi-test execution
- stronger SQL generation

# Evaluation Layer

## Written Questions

## Coding / SQL
